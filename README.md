# MindFlow (마인드플로우) - 브레인스토밍 메모 보드
> **MindFlow**는 빈 캔버스 위에 아이디어 메모 카드를 자유롭게 생성하고, 관련 있는 메모를 실시간으로 연결하여 구조화할 수 있는 **순수 정적(Pure Static) 브레인스토밍 웹 애플리케이션**입니다.
Miro나 Figma와 유사한 무한 캔버스 이동 및 줌 인터랙션을 제공하며, 세련된 다크 테마와 글래스모피즘(Glassmorphism) 스타일로 프리미엄한 사용자 경험을 선사합니다.
---
## 🎨 주요 기능 (Features)
1. **무한 캔버스 조작 (Pan & Zoom)**
   - 마우스 우클릭 / 휠 클릭 / `Space + 드래그`를 통해 넓은 캔버스를 제한 없이 탐색할 수 있습니다.
   - 마우스 포인터의 위치를 기준으로 한 부드러운 확대/축소(Zoom)를 지원합니다 (0.15배 ~ 3.0배).
   - 캔버스 줌 레벨에 연동되는 반응형 도트 그리드 배경이 적용되었습니다.
2. **메모 카드 (노드) 조작**
   - 캔버스를 **더블 클릭**하거나 왼쪽 툴바의 **[+]** 단추를 눌러 즉시 새 메모를 추가합니다.
   - 작성하는 텍스트 내용에 맞게 메모 카드의 높이가 실시간으로 늘어납니다.
   - 드래그 앤 드롭으로 자유롭게 배치할 수 있으며, 그리드 스냅(Snap to Grid) 옵션을 활성화하여 칼각 배치가 가능합니다.
   - 슬레이트, 로즈, 에메랄드, 앰버, 오션, 바이올렛의 6가지 감각적인 반투명 글래스 테마 컬러를 제공합니다.
3. **메모 연결 (동적 베지에 곡선) 및 수정**
   - 메모 가장자리의 4방향 포트(원형 단추)에서 드래그하여 다른 메모 카드에 드롭하면 아름다운 **3차 베지에 곡선(Bezier Curve)** 연결선이 이어집니다.
   - 드롭 위치에 관계없이 자동으로 가장 가까운 포트를 찾아 연결선을 정렬합니다.
   - **연결선 즉시 수정 (New)**: 연결선을 마우스로 선택하면 활성화되는 양 끝단의 **수정용 핸들**을 드래그하여 다른 포트로 언제든 쉽게 재연결할 수 있습니다.
   - 연결선을 선택하고 중간의 삭제 버튼이나 `Delete`/`Backspace` 키를 통해 손쉽게 선을 지울 수 있습니다.
4. **히스토리 제어 (Undo / Redo)**
   - 메모 추가/이동/삭제/편집, 연결선 생성/삭제/수정 등 모든 작업에 대해 50단계 깊이의 **되돌리기(Undo: `Ctrl+Z`) 및 다시 실행(Redo: `Ctrl+Y`)** 스택을 내장하고 있습니다.
5. **실시간 메모 검색 및 포커싱**
   - 상단 검색창에서 메모 텍스트를 실시간으로 검색하여 일치하는 모든 메모를 강조합니다.
   - `Enter` 키를 눌러 매칭되는 메모들로 캔버스를 부드럽게 스냅하고 즉시 줌 포커싱합니다 (`Ctrl+F` 단축키 제공).
6. **로컬 자동 저장 & 데이터 가져오기/내보내기**
   - 모든 캔버스 상태(좌표, 줌 배율, 메모 텍스트, 좌표, 색상, 연결 정보)가 브라우저의 `LocalStorage`에 실시간 자동 저장되어 브라우저를 닫아도 복구됩니다.
   - `.json` 형식으로 내 데이터를 안전하게 저장(내보내기)하거나 불러와서(가져오기) 작업할 수 있습니다.

## ⌨️ 단축키 안내 (Shortcuts)
| 단축키 | 설명 |
| :--- | :--- |
| **더블 클릭 (Double Click)** | 마우스 커서 위치에 새 메모 카드 생성 |
| **오른쪽 드래그 / 휠 클릭 드래그** | 캔버스 화면 이동 (Pan) |
| **Space + 왼쪽 드래그** | 캔버스 화면 이동 (Pan) |
| **마우스 휠 스크롤** | 커서 기준 화면 확대 / 축소 (Zoom) |
| **`N`** | 화면 중앙에 새 메모 생성 |
| **`G`** | 그리드 스냅 (Snap to Grid) 토글 |
| **`Ctrl` + `Z`** | 이전 작업 되돌리기 (Undo) |
| **`Ctrl` + `Y`** | 되돌린 작업 다시 실행 (Redo) |
| **`Ctrl` + `F`** | 메모 검색창 포커스 및 강조 |
| **`Delete` / `Backspace`** | 선택된 메모 또는 선택된 연결선 삭제 |
| **`Esc`** | 선택 해제 / 모달 닫기 / 작업 취소 |

## 🛠️ 기술 스택 (Tech Stack)
- **Markup & Structure**: HTML5 (시맨틱 웹 표준 준수)

## 🚀 실행 방법 (How to Run)
본 프로젝트는 외부 데이터베이스나 별도의 백엔드 API 연동이 필요 없는 **100% 정적 프론트엔드 프로젝트**입니다.
### 방법 1. index.html 직접 실행
1. 저장소 파일들을 로컬 폴더에 다운로드합니다.
2. [index.html](file:///c:/Users/aeter/Documents/sw-final-folder/index.html) 파일을 더블 클릭하여 크롬, 엣지, 사파리 등의 브라우저로 즉시 실행합니다.
### 방법 2. 로컬 웹 서버 실행 (추천)
로컬 웹 서버 환경에서 구동하면 보안(CORS)이나 일부 브라우저의 File URL 정책 제한을 우려할 필요 없이 가장 완벽하게 작동합니다.
* **Python 3 환경인 경우:**
  ```bash
  python -m http.server 8000
  ```
  이후 브라우저에서 `http://localhost:8000/` 로 접속합니다.
* **Node.js 및 npm이 설치된 경우:**
  ```bash
  npx http-server -p 8000
  ```
  이후 브라우저에서 `http://localhost:8000/` 로 접속합니다.
