MindFlow 사용법

핵심 기능

  • 메모 생성: 빈 화면을 더블 클릭하거나 왼쪽 툴바의 [+] 아이콘을 누릅니다.
  • 화면 이동(Pan): 마우스 오른쪽 버튼 또는 휠 클릭한 상태로 드래그하거나, 스페이스바를 누른 채 왼쪽 마우스 드래그를 합니다.
  • 화면 줌(Zoom): 마우스 휠 스크롤을 하거나 우측 하단 줌 패널의 +, - 버튼을 클릭합니다.
  • 메모 연결: 각 메모의 모서리에 마우스를 대면 활성화되는 커넥터 포트(원형 단추)에서 다른 메모 카드로 드래그 앤 드롭하면 화살표 연결선이 이어집니다.
  • 메모/연결 삭제: 메모의 우측 상단 [x] 버튼을 누르거나, 연결선을 클릭한 후 표시되는 삭제 버튼을 누릅니다.

단축키 안내

Double Click 새 메모 생성
Space + Drag 화면 이동
N 새 메모 생성
G 그리드 스냅 전환
Ctrl + Z 실행 취소 (Undo)
Ctrl + Y 다시 실행 (Redo)
Ctrl + F 메모 검색창 열기
Esc 모달 닫기 / 작업 취소
